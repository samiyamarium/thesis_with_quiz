Core Agent Factory Concepts

Two Modes of General Agent Use

General Agent Mode: Solve tasks directly.

Factory Mode: Create reusable assets, workflows, prompts, evaluations and organizational knowledge.

Seven Invariants

Core principles that remain stable regardless of implementation, models, tools or platforms.

These ensure reliability and repeatability.

Reference Stack

A structured architecture describing how agents operate.

Includes governance, memory, context engineering, evaluation and runtime execution.

Context Engineering

The science of managing information flow into an agent.

High quality context leads to high quality outputs.

Memory Systems

Memory provides continuity across sessions and tasks.

It transforms isolated interactions into cumulative intelligence.

Evaluation Loops

Measure agent quality and performance.

Evaluation drives continuous improvement and stable production systems.

Stable Systems

Stable systems remain predictable under changing conditions.

Observability and measurement are critical characteristics.

Asset Accumulation

Every successful task should become a reusable organizational asset.

This creates compounding productivity gains.
